# ml-data-pipeline(пре альфа  версия)
##Прототип проекта:  Инструмент предобработки данных и восстановления пропусков с GUI  

## Задача:
Создать  инструмент для:
- подготовки датасетов
- обучения модели на подготовленном датасете
- тестирования  инференса обученной модели

## Что сделано:

— Разработал приложение на Python с графическим интерфейсом (PySide6) для обработки датасетов:  
      • Очистка строковых признаков, хеширование, кодирование (One-Hot, Label, Target и др.)  
      • Балансировка, обрезка датасета(SMOTE,TRIM)      
      • Восстановление пропущенных значений с помощью обученных моделей (sklearn)  
      • Версионирование изменений через мета-систему (# META:)  
      • Анализ и визуализация корреляции параметров.
      • Моделирование  и тест инференса модели
   
## Результат:

Разработал  end-to-end инструмент от обработки и анализа датасетов до обучения и инференса модели.
— Реализована поддержка .pkl/.joblib моделей, динамическое обновление состояния  
— Инструмент позволяет специалисту повторно использовать обработку, отслеживать изменения и быстро экспериментировать  
— Технологии: Python, Pandas, Scikit-learn, PySide6, joblib, category_encoders, GUI
